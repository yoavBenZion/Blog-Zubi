Callback Functions

מהו callback?

callback הוא משתנה שהועבר לתוך פונקציה , מתישהו במהלך הפעלת הפונקציה , נריץ את המשתנה callback שמקבל פונקציה כלשהי ובכך ניצור אינטגרציה בין שתי פונקציות. בקלות אנחנו יכולים להעביר קוד מפונקציה אחת לתוך פונקציה אחרת.

בג'אווה סקריפט אם נמתין לפונקציה שתסיים ותחזיר תשובה אנחנו עלולים סתם לתקוע את הדפדפן לבקשות אחרות, לכן אנחנו משתמשים בcallback כדי לבצע פעולות באופן אסינכרוני.
מדובר בפתרון יעיל ומאוד שימושי עבור פעולות שיכולות לקחת קצת יותר זמן מהרגיל, לדוגמה בקשות באמצעות AJAX.

על מנת להבין מהו callback נתחיל בדוגמה פשוטה ביותר באמצעות פונקציה setTimeout :

setTimeout(callback,delay);
המשתנה callback מקבל פונקציה לתוך setTimeout, אחרי x שניות הפונקציה שהועברה כמשתנה תופעל.

דרך פשוטה יותר כדי להמחיש כמו שצריך את הנושא בפשטות - להשתמש בפונקציה אנונימית (פונקציה שניתן להשתמש בה מבלי שנגדיר לה שם. פונקציה אנונימית נוצרת וקוראים לה פעם אחת):

setTimeout(function(){
   ,alert("this is anonymous function")},delay);
הפונקציה האנונימית תופעל לאחר x שניות ויוצג חלון אזהרה: "this is anonymous function”

על הדרך מי שמתעניין לחקור קצת יותר לגבי ההבדלים ומה כדאי להשתמש settimeout או setinterval מוזמן להיכנס לדיון הבא בstackoverflow (מומלץ לעיין בתשובה מס' 2)


נתקדם לדוגמה נוספת, נבנה סה"כ 3 פונקציות:

  • - פונקציה שמבצעת פעולת חיבור בין 2 מספרים ומחזירה את התוצאה.
  • - פונקציה שמבצעת פעולת כפל בין 2 מספרים ומחזירה גם היא את התוצאה.
  • - פונקציה שמקבלת 3 משתנים:
    • + משתנה 1: מספר ראשון
    • + משתנה 2: מספר שני
    • + משתנה 3: callback - משתנה זה מקבל פונקציה שיודעת לעשות חישוב מתמטי – במקרה שלנו חיבור או כפל.

נתחיל עם הפונקציות הבסיסיות ביותר שהגדרנו למעלה פעולות החיבור והכפל

let add = function (a,b){ return a+b;} ;
let multiply = function (a,b) { return a*b;};

אחרי שבנינו שתי פונקציות בסיסיות שיודעות לעשות את הפעולות המתמטיות הפשוטות חיבור וחיסור, נעבור לפונקציה השלישית שהגדרנו למעלה המקבלת 3 משתנים:

נבנה פונקציה שתקבל 2 מספרים (כל מספר הוא משתנה) ולקרוא לפעולה (משתנה-callback function) שבה נרצה להשתמש – כפל וחיבור:

let calc = function (num1, num2, callback){
   return callback(num1,num2);};

כעת נרצה לבצע פעולת חישוב בסיסית ביותר בעזרת 2 משתנים ואותה נדפיס בconsole, על מנת שנוכל ליצור אינטגרציה בין פונקציית החישוב (calc)לפעולה המתמטית נצטרך להציב בתוך משתנה הcallback את הפעולה add או multiply:

כך זה יראה:

console.log(calc(2,3,add));// will print 5
console.log(calc(2,3,multiply)); // will print 6 in console


לפני שאציג איך מיישמים בקשת AJAX בעזרת callbacks, מי שאין לו רקע בכלל בjquery ובעיקר בAJAX רצוי לקרוא קצת לפני ולנסות להבין על מה מדובר, למרות שהקוד כלל לא מורכב וסביר מאוד שכן תצליחו להבין.

בדוגמה הבאה נבנה קודם את הפונקציות שנשתמש בהן כcallbacks ולאחר מכן ניישם אותן בקריאת AJAX:


Var someUlr = …;
function successCallback(){
//success code
}
function completeCallback(){
//complete code
}
function errorCallback(){
//error code
}
$.ajax({
url: someUrl,
success: successCallback,
complete: completeCallback,
error: errorCallback
})

נסביר מה עשינו כאן:
בנינו 3 פונקציות שהשתמשנו בהן בתוך בקשת הAJAX, במידה והבקשה מצליחה וקיבלנו תגובה מהשרת (קוד 200) יופעל ()successCallback במידה ויש בעיה נקבל תגובה מהשרת (לדוגמה שגיאה 404 דף לא נמצא) אך בכל מקרה ברגע שנסיים את הבקשה יופעל ()completeCallback. ניתן לראות שהקוד נורא מאורגן ומסודר, יותר פשוט מזה אין!
למי שרוצה להרחיב על AJAX הכל נמצא כאן

עכשיו כשאתם יודעים קצת יותר מהו callback פתחו קוד שרשמתם לאחרונה ותנסו לראות איך אתם הופכים את הקוד ליותר יעיל בעזרת callback ובעיקר שאתם לא חוזרים על עצמכם בכתיבת הקוד (DRY- Do not repeat yourself) !

callback function ילווה אתכם תמיד בין אם בחרתם במסלול front end ובין אם התקדמתם ל back end כך שחשוב מאוד להבין את הנושא לעומק!
