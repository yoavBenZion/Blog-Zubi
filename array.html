Array Cheat Sheet

Uses & Methods

מאמר זה עוסק במתודות של ARRAY.

המאמר נועד למפתחים מתחילים שעושים את הצעדים הראשונים שלהם בעולם ה Front End או למתקדמים אשר רוצים לרענן את הזיכרון שלהם בנושא.

Array – נחשב אחד מאבני היסוד (core), של שפת JavaScript. ואי אפשר באמת להבין את העומק שלה, בלי לשלוט בנושא. כמו כן, מאוד מקובל להישאל בראיון עבודה (במיוחד בשלבים הראשונים של כניסה לתחום), מה עושה מתודת array מסוימת.

אני ממליץ בחום לקרוא את המאמר הזה לפחות פעם אחת בחייכם, ולחזור לפה מידי פעם, כדי לרענן את הידע שלכם.
הרי ידוע שמה שלא משתמשים בו ביום יום, הולך לאיבוד (:

ניתן להשתמש בחלונית החיפוש למטה, ע"י כתיבת המתודה המבוקשת, ולחיצה על כפתור חיפוש.

מקרא :

no change
- המתודה לא משנה את המערך המקורי.

change
- המתודה משנה את המערך המקורי.

ES5
- המתודה היא חלק מהפיתוחים שנכנסו במסגרת ES5 .

» undefined //When you declare an Array, you will allways get back undefined.

↓ Serch BOX below ↓

  1. ()isArray
    no change

    Returns True or False.

    ☝︎

    var boo = [1,2,3];
    » undefined
    boo;
    » (3) [1,2,3]
    Array.isArray(boo);
    » true
    Array.isArray();
    » false
    Array.isArray('string');
    » false

  2. Length
    no change

    Returns the number of elements in the Array.

    מחזיר את מס' האלמנטים במערך.

    var boo = [1,2,3, 'someString', true];
    » undefined
    boo;
    » (5) [1,2,3, 'someString', true]
    boo.length;
    » 5

  3. ()indexOf
    no change

    מחזיר את ערך ה index של הפרמטר המועבר במתודה, אם הערך אינו נמצא במערך, מחזיר 1- .

    דוגמא א :
    המתודה תחזיר את ה index value , הראשון של האלמנט שמועבר במתודה, (בתוך הסוגריים) .
    הספירה תתחיל מ index 0, אלא אם כן יועבר פרמטר נוסף.

    var boo = [1,2,3,1,2,3];
    » undefined
    boo;
    » (6) [1,2,3,1,2,3]
    boo.indexOf(3);
    » 2

    דוגמא ב :
    הפרמטר השני המועבר במתודה, קובע את ה index ממנו תתחיל הספירה.

    boo;
    » (6) [1,2,3,1,2,3]
    boo.indexOf(3,3);
    » 5

    דוגמא ג :
    במידה והפרמטר המועבר במתודה לא קיים במערך, נקבל בחזרה 1-.

    boo;
    » (6) [1,2,3,1,2,3]
    boo.indexOf(6);
    » -1

  4. ()lastIndexOf
    no change

    מחזיר את ערך ה index בו האלמנט המועבר במתודה נמצא, כאשר הספירה מתחילה מסוף המערך .

    boo;
    » (6) [1,2,3,1,2,3]
    boo.lastIndexOf(2);
    » 4

  5. ()push
    change

    מכניס ערכים (אחד או יותר), לסוף המערך
    מחזיר את ה length של המערך.

    var boo = [1,2,3];
    » undefined
    boo.push(4,5);
    » 5
    boo
    » (5) [1,2,3,4,5]

  6. ()pop
    change

    מוריד אלמנט מסוף המערך,
    מחזיר את ה Value של האלמנט שהוצא
    אם המערך היה ריק, מחזיר Undefined.

    var boo = [1,2,3];
    » undefined
    boo.pop();
    » 3

  7. ()unshift
    change

    מוסיף ערכים (אחד או יותר), לתחילת המערך,
    מחזיר את ה length של המערך .

    var boo = [1,2,3];
    » undefined
    boo.unshift(0);
    » 4
    boo;
    » (4) [0,1,2,3]

  8. ()shift
    change

    מוציא ערך מתחילת המערך,
    מחזיר את Value של האלמנט שהוצא .
    במידה והמערך ריק, מחזיר Undefined.

    var boo = [1,2,3];
    » undefined
    boo.shift();
    » 1
    boo
    » (2) [2,3]

  9. ()toString
    no change

    מחזיר את כל המערך כ string מופרד בפסיק.
    לא מקבל פרמטרים, או משנה את המערך .

    var boo = [1,2,3];
    » undefined
    boo.toString();
    » "1,2,3"

  10. ()join
    no change

    מחזיר את המערך כ string. כאשר הפרמטר המועבר בסוגריים, יהיה המפריד של ה string.
    אם משתמשים במתודה ללא פרמטר, הערכים יחזרו כ string, מופרד ע"י פסיק .

    ניתן גם להחליף את הפסיקים בסמן אחר, לדוגמא : -

    var boo = [1,2,3];
    » undefined
    boo.join(); // Default.
    » "1,2,3"
    boo.join(''); // No separator.
    » "123"
    boo.join(-); // Changing the separator.
    » "1-2-3"

  11. ()reverse
    change

    משנה את המערך במקום, ומחזיר אותו הפוך reversed.
    בתוך המערך הישן.

    var boo = [1,2,3];
    » undefined
    boo.reverse();
    » (3) [3,2,1]
    boo
    » (3) [1,2,3]

  12. ()sort
    change

    משנה את הערכים במקום ומחזיר את ה Sorted array
    שים לב!
    המתודה מסדרת את המערך לפי unicode, שלמעשה מסדרת את המערך לפי סדר אלפביתי.
    למעשה, כדי לסדר את המערך בסדר אחר, עולה-יורד/ יורד-עולה, יש להשתמש ב
    Compare Function כפרמטר המועבר בפונקציה.

    var boo = ['zebra', 'lion', 'fish'];
    » undefined
    boo.sort();
    » (3) ["fish", "lion", "zebra"]
    var foo = [11,20,6,54]
    » undefined
    foo.sort();
    » (4) [11,20,54,6]

    דוגמא לפונקציות השוואה מגבוה לנמוך ולהפך

    var foo = [11,20,6,54];
    » undefined
    boo.sort();
    »(4) [11,20,54,6]
    // Smallest to highest
    foo.sort((a,b) => {return a-b})
    » (4) [6,11,20,54]
    // Highest to Smallest
    foo.sort((a,b) => {return b-a})
    » (4) [54,20,11,6]

  13. ()concat
    no change

    מייצר מערך חדש, ע"י חיבור פרמטרים שהועברו במתודה,
    למערך עליו המתודה נקראה, מחזיר מערך חדש.

    var boo = [1,2,3];
    » undefined
    var foo = boo.concat('one', 'two');
    » undefined
    foo
    » (5) [1,2,3,"one", "two"]
    boo
    (3) [1,2,3]

  14. ()slice
    no change

    מייצר העתק של המערך המקורי .
    מחזיר Slice of the array.

    הפרמטר הראשון שמועבר במתודה, קובע את מיקום תחילת המערך החדש, ויהיה כלול בו .
    הפרמטר השני המועבר במתודה, קובע את סוף החיתוך, אך לא יהיה כלול במערך החדש.

    במידה ומעבירם רק פרמטר אחד, ממנו המערך החדש ייגזר, עד סוף המערך הישן.
    פרמטר שלילי קובע שהחיתוך יתבצע מסוף המערך .
    כאשר 1- יהיה המיקום האחרון במערך.

    var boo = ['one', 'two', 'three', 'four', 'five'];
    » undefined
    boo.slice(1);
    » (4) ['two', 'three', 'four', 'five']
    boo.slice(1,3)
    » (2) ['two', 'three']
    boo.slice(-2)
    » (2) ['four', 'five']
    boo
    » (5) ['one', 'two', 'three', 'four', 'five']

  15. ()splice
    change

    במתודה זו משתמשים לשינוי המערך ע"י הוספה או הסרה של אלמנטים מהמערך המקורי.

    בשונה מ pop או shift, ב splice, ניתן להשתמש כדי להוציא כל אלמנט, בכל אינדקס.

    הסרה

    כאשר מסירים אלמנטים מהמערך,
    יחזרו ה Removed elments.

    הפרמטר הראשון מייצג את ה index positin
    ממנו יתחילו להיחתך ערכים מהמערך.
    הפרמטר השני, מפרט מה מס'' האלמנטים שיש להסיר.

    var boo = ['one', 'two', 'three', 'four', 'five'];
    » undefined
    boo.splice(0,2);
    » (2) ["one", "two"]
    boo
    » (3) ["three", "four", "five"]

    הוספה

    כל פרמטר שיעבר לאחר הפרמטר השני, יועבר כאלמנט למערך החדש.
    במידה והפרמטר השני הוא 0, שום אלמנט לא יוסר מהמערך.
    במידה ושום אלמנט לא מוסר, חוזר מערך ריק כ - Empty Array.

    var boo = ['one', 'two', 'three', 'four', 'five'];
    » undefined
    boo.splice(2,0,1,2,3);
    » []
    boo
    » (8) ["one", "two", 1, 2, 3, "three", "four", "five"]

  16. ()forEach
    ES5
    change

    מפעיל callback function שמועברת כפרמטר במתודה, על כל אחד מהאלמנטים במערך מחזיר undefined.

    פונקציית Callback.
    מקבלת שלושה פרמטרים - Value, index & Array.

    var foo = [1,2,3,4,5];
    » undefined
    foo.forEach(function(v,i,a) { a[i] = v+1});
    » undefined
    foo
    » (5) [2,3,4,5,6]

  17. ()map
    ES5
    no change

    מחזיר מערך חדש, ע"י הפעלת פונקציה על כל אחד מהאלמנטים במערך הישן.

    var foo = [1,2,3,4,5];
    » undefined
    foo.map((x)=>{ return x*x});
    » (5) [1,4,9,16,25]

  18. ()reduce
    ES5
    no change

    מפעילה פונקציה על כל אחד מהאלמנטים במערך
    משמאל לימין, בצורה מצטברת.
    מקטינה את המערך לכדי singel Value.

    var foo = [1,2,3,4,5];
    » undefined
    foo.reduce((x,y)=> { return x+y});
    » 15
    foo.reduce((x,y)=> { return x+y},6);
    » 21

    שים לב! בשונה ממתודות ES5 אחרות
    Reduce callback function, מקבלת 4 פרמטרים :

    Accumulater,
    Current Value,
    index,
    & Array.

    optional
    אופציה נוספת היא להשתמש בערך התחלתי , כפרמטר שני המועבר עם הפונקציה שישמש כערך צובר התחלתי לחיבור האלמנטים.

  19. ()reduceRight
    ES5
    no change

    בדומה (אך הפוך), למתודה הקודמת - reduce,
    מפעילה פונקציה על כל אחד מהאלמנטים במערך
    מימין לשמאל, בצורה מצטברת.
    מקטינה את המערך לכדי singel Value.

    var boo = ['a','b','c','d'];
    » undefined
    boo.reduceRight((x,y)=> { return x+y});
    » "dcba"
    boo.reduce((x,y)=> { return x+y})
    » "abcd"

  20. ()filter
    ES5
    no change

    המתודה מפעילה פונקציה, על כל אחד מהאלמנטים של המערך,
    ובודקת אם האלמנט עובר את המשוואה שמועברת בפונקצית ה
    callback function.
    מחזיר מערך חדש, שמכיל רק את האלמנטים שעומדים במשוואה, ומחזירים ערך True לפונקציה שהועברה.

    var foo = [10, 22, 13, 44, 52, 23];
    » undefined
    foo.filter((x)=> { return x < 20});
    » (2) [10,13]
    foo.filter((x)=>{ return x > 20});
    » (4) [22,44,52,23]
    foo
    » (6) [10,22,13,44,52,23]

  21. ()every
    ES5
    no change

    מחזיר True או False.

    הפונקציה רצה על כל אלמנט במערך, ובודקת אם כל האלמנטים עומדים במשוואה שמועברת בה.
    עוצרת מיד כשפוגשת ערך, שמחזיר לה false .

    var foo = [10, 22, 13, 44, 52, 23];
    » undefined
    foo.every((e)=>{ return e > 50;})
    » false
    foo.every((e)=>{ return e < 60;})
    » true
    foo
    » [10, 22, 13, 44, 52, 23]

  22. ()some
    ES5
    no change

    מחזיר True או False.

    עוברת על כל המערך, ובודקת האם יש אלמנט אחד בלבד שעונה לדרישת המשוואה שהוכנסה לפונקציה,
    במידה וכן, המתודה מפסיקה את הבדיקה של המערך - ומחזירה true .
    אחרת, במידה והריצה הגיעה לסיום המערך - יחזור false.

    var foo = [10, 22, 13, 44, 52, 23];
    » undefined
    foo.some((e)=>{ return e > 15;})
    » true
    foo.some((e)=>{ return e < 5;})
    » false

    שים לב! כאשר המתודות האלו מופעלות על מערך ריק, יש להם התנהגות קבועה.
    ()some מחזיר False,
    ו()every מחזיר True.

    var boo = [];
    » undefined
    boo.some((e)=>{ return e > 5;};
    » false
    boo.every((e)=>{ return e > 5;}
    » true
