JavaScript and the DOM

או למה את מתכוונת כשאת אומרת DOM ?

ה DOM או Document Object Model ב JavaScript , הוא אחד הכלים הכי חזקים איתם נעבוד בפיתוח Front End .

למעשה, את כל האינטראקטיביות של הדף, כגון שינויים בעיצוב כתלות בפעולות המשתמש כגון קליק, הזזה של העכבר או פתיחה של חלון חדש לפי זמן קבוע, כולן פעולות המבוססות על מניפולציות של JS על הDOM.

כדי להתחיל לדבר על ה DOM , אכתוב ממש בקצרה מהו אובייקט ב JS :
Object - הדרך הנכונה להסתכל על אובייקטים, זה על פרט שיש לו Property & Methods. כאשר :

  • Property: ערך משתנה, ששייך לאובייקט.
  • Method: משהוא שאפשר לעשות עם/ל/על אובייקט.

לדוגמא :
Array הוא אובייקט, יש לו property כמו length ויש לו method, כמו ()push שמוסיף לו ערכים.

משתנים גלובליים

בדפדפן יש כמה משתנים גלובליים מובנים, איתם אנחנו יכולים לעבוד.
כל המשתנים האלו, לדוגמא ()alert או ()location , הם properties של ה window .

window object - נמצא בפסגת ההיררכיה של העמוד, עליו יושבים כל המשתנים הגלובליים שאנחנו מכירים, וגם האובייקט document . Document object – כל עמוד html שנטען בדפדפן, הופך ל Document object . ומכיל את כל תוכן העמוד.

למעשה ב DOM, כל אלמנט html הוא אובייקט, וכל ערכי ה CSS הם חלק מה property value pairs שנמצאים ב property style של אותו אובייקט.

DOM – the virtual map of the webpage

Document object model - המודל מתייחס לדיאגרמה (הציור), של מבנה הדף . כאשר כל אלמנט html , מקבל מקום משלו על העץ. מיקום האלמנטים בעץ, מוגדר עפ"י הסדר (היררכיה) של ה html.

אנסה לפשט את זה ע"י הציור הבא :

כשמשווים את הדיאגרמה משמאל, ל html Syntax מימין. אפשר לראות שכל אלמנט תופס את מיקומו ההיררכי ב DOM . החלק האפור – מסמן את הצד הוויזואלי של ה webpage . ב body יושבים האלמנטים H1 , P, UL . כשכל אלמנט שווה מבחינת סדר היררכי, אך המיקום שלו בעמוד מצויר משמאל לימין. ומתחת ל UL , בהמשך לסדר ההיררכי אפשר לראות 3 אלמנטים של LI .

הציור של ה DOM , מזכיר מבנה של עץ הפוך, ולכן מקובל לקרוא לדיאגרמה הזו
The DOM tree .

The DOM tree

המבנה של העץ לקוח מעולם מדעי המחשב ומנסה לתאר את המורכבות של הקשרים בין האלמנטים. אפשר להסתכל על המבנה של אילן יוחסין, המתאר את הקשר בין ה parent (לדוגמא ה body ), ל child (לדוגמא ה p ).

אבל מה נותן ה DOM ?

למעשה, מרגע שיש לנו מפה היררכית של מבנה העמוד. אנחנו יכולים להגיע לכל אלמנט בעמוד, ולבצע בו שינויים או פעולות בעזרת JavaScript .
אז באיזה אלמנטים אפשר לבצע שינויים ?

    NODE - הוא למעשה כל דבר שניתן לשנות, כגון :

  • 1. Html elements
  • 2. Text
  • 3. Attributes – href / class / style וכו' ..

לדוגמא האלמנט li הוא node, והטקסט שבתוכו נחשב node נפרד. וכך אפשר לבחור את ה node הספציפי שאותו אנו רוצים לשנות.

Interacting with the web page

    ה DOM, הוא הבסיס לאינטראקציה עם הדפדפן ,על ידי ביצוע 3 פעולות מרכזיות :

  • 1. בחירת אלמנטים בדף. (Selecting ).
  • 2. ביצוע שינויים באלמנטים. (manipulating ).
  • 3. הקשבה לפעולות המשתמש בדף. (Listening ) .

בחירת אלמנטים בדף
(Selecting) :

ניקח את הקוד html הבא כדוגמא :

אותו נבחר בעזרת javaScript


document.querySelector('.header');

ונקבל את הפלט הבא ב console

באותה צורה אפשר לקבל רשימת ערכים של רשימת LI כ ARRAY , לדוגמא :


document.querySelectorAll('li');
>(3)[li,li,li]
document.querySelectorAll('li')[0];
> This will give us the first node of the li tag.

ביצוע שינויים באלמנטים
(manipulating) :

    אלו רק כמה דוגמאות לאפשרויות שנוכל לעשות עם JS :

  • 1. CreateElement – מאפשר לנו ליצור אלמט חדש.
  • 2. textContent – מאפשר להכניס בו text .
  • 3. appendChild - מאפשר לצרף אותו לאלמנט קיים כ child .
//creating an element
var para = document.createElement('p');
//creating a text node in the element
para.textContent = 'This is a color app, try to move your courser on the screen';
//apending the new element to the body
document.body.appendChild(para);

יש עוד המון שינויים שאפשר לעשות לכל אלמנט כולל שינוי ה style שלו, כל מה שצריך זה לזכור שכל אלמנט בעמוד הוא אובייקט, ולכל אובייקט יש property value pairs . אחד מהproperties האלו – הוא style שתחתיו נמצאים כל ערכי ה CSS .

//adding style
para.style.backgroundColor = 'lightgreen'

הקשבה לפעולות המשתמש בדף (Listening) :

בעזרת JS אפשר להצמיד event לכל אלמנט בעמוד . לדוגמא, הוספת event מסוג mousemove יכול לשמור לנו את הקורדינטות של העכבר כל פעם שאנחנו מעבירים את העכבר על האלמנט הנבחר. למטה, אני משתמש בקורדינטות האלו, כדי לשנות את צבע הרקע של העמוד לפי ()rgb .

//creating an element
var div = document.createElement('DIV');
//setting it's height
div.style.height = '100vh';
// appending it to the page
document.body.appendChild(div);
// adding a mousemove Event listner
div.addEventListener('mousemove', function (e){
var x = e.clientX;
var y = e.clientY;
div.textContent = ' R=' + x +' ' + ' G=' + y + ' B=' + (x-y);
div.style.textAlign= 'center';
div.style.backgroundColor = 'rgb('+ x + ',' + y + ',' + (x-y) +')';
});

Traversing:

החלק האחרון של המאמר יעסוק במעבר בין אלמנטים בעמוד, שזה חלק חשוב בשימוש של ה DOM, כבר ראינו למעלה שאנחנו יכולים לקבל מס' אלמנטים כמערך, אבל אנחנו גם יכולים לבחור באלמנט מסוים ע"י בחירה של קרוב משפחה שלו, כמו sibling או parent .
מה זה נותן בעצם ?

הרי אנחנו יודעים למצוא כל אלמנט בעמוד בעזרת querySelector ?

אז למעשה, אחרי שמצאנו אלמנט אחד בעמוד, אנחנו לא צריכים לפנות כל פעם ל document object , אנחנו פשוט יכולים להשתמש ב קרבה המשפחתית שלו להשיג אלמנט קרוב.
וכך אפשר לרשום קוד קצר יותר, שימושי יותר, ופונקציות יעילות יותר.

אפשר לראות בציור הבא, שלכל אלמנט בעמוד יש אלמנט קרוב אליו בקרבה משפחתית (לפי האילן יוחסין).

לסיום, מקווה שנתתי לכם מספיק רעיונות ודוגמאות לכלים שימושיים שאפשר לעשות עם ה DOM , ואפילו קצת חשק לקרוא עוד קצת חומר בנושא כדי שתוכלו להשתמש בכלי הזה כמו שצריך בפרויקטים שלכם בעתיד.
